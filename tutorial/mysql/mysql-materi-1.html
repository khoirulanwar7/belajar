Tutorial Cara Install XAMPP

Kumpulan tutorial belajar MySQL untuk pemula, dari dasar hingga mahir.

Xampp merupakan paket perangkat lunak digunakan untuk memudahkan instalasi dan penggunakan web server. XAMPP terdiri dari beberapa perangkat lunak antara lain: Apache (web server), MariaDB/MySQL (database), PHP dan Perl (bahasa pemrograman).

X = Cross Platform

A = Apache

M = MariaDB / MySQL

P = PHP

P = Perl

Pada artikel ini akan dibahas cara instalasi Xampp di Windows dan Linux

Cara Install XAMPP di Windows

Tahap 1, Download File Installer

Download Installer XAMPP di halaman resmi Apache Friends dan pilih versi Linux (XAMPP for Windows)

Halaman official XAMPP

Tahap 2, Install XAMPP

Buka Folder Download dan double click installer XAMPP

Halaman setup akan muncul. Tekan Next

Tekan Next untuk melanjutkan

Lokasi instalasi default di C:/xampp. Tekan Next

Tunggu proses install sampai selesai selanjutnya tekan Finish

Tahap 3, Jalankan XAMPP

Buka control panel di menu aplikasi. Jalankan service Apache dan MySQL

Buka browser dan ketik

http://localhost
