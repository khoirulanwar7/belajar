Tutorial Pemrogaman Java untuk Pemula

Kumpulan tutorial belajar Java untuk pemula, dari dasar hingga mahir.

Operator

1. Operator Aritmatika

Operator yang berfungsi untuk operasi aritmatika
(+) Plus
(-) Minus
(*) Point
(/) Divide
(%) Modulus

2. Operator Assignment (penugasan)

= (Untuk mengisi nilai)
+= (Untuk mengisi nilai)
-= (Untuk mengisi dan melakukan operasi pengurangan)
*= (Untuk mengisi dan melakukan operasi perkalian)
/= (Untuk mengisi dan melakukan operasi pembagian)
%= (Untuk mengisi dan melakukan operasi sisabagi)

3. Operator Increment dan Decrement

Format penulisan : (tipe data) namaVariabel ; Operator increment dan decrement digunakan untuk menaikkan atau menurunkan suatu nilai integer sebanyak suatu nilai integer, sebanyak satu satuan

++ (Menambahkan 1 nilai)
-- (Mengurangi 1 nilai)

4. Operator Relasi (Pembanding)

> (Lebih besar)
< (Lebih kecil)
== (Sama dengan)
!= (Tidak sama dengan)
>= (Lebih besar sama dengan)
<= (Lebih kecil sama dengan)

5. Operator Logika

&& (Logika AND) Apabila pernyataan 1 dan 2 true, maka kode program akan bernilai true.
|| (Logika OR) Apabila salah satu pernyataan bernilai true, maka kode program akan bernilai true.
! (Negasi atau Kebalikan) Apabila pernyataan bernilai true, maka kode program akan bernilai false begitupun sebaliknya.
