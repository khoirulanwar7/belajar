Variabel dan Tipe Data

Variabel adalah tempat menyimpan nilai sementara yang nantinya akan digunakan kembali, terbagi menjadi dua, yaitu variabel lokal dan variabel global.Variabel lokal adalah variabel yang hanya bisa dikenali pada sub program, sedangkan variabel global dapat dikenali pada keseluruhan program. Dalam variabel terdapat tipe data dan nama variabel.

Tipe data

adalah jenis data yang tersimpan dalam variabel

  • char : karakter (contoh : ‘A’)
  • int, short, byte,long : angka atau bilangan bulat (contoh : 1 dan 23)
  • float, double: bilangan desimal (contoh :2.4)
  • String : teks (contoh : “rumah”, “sekolah”)
  • boolean : tipe data yang nilainya hanya berupa true dan false

Penulisan Variabel

Format penulisan :
(tipe data) namaVariabel;
Contoh :

  1. Membuat deklarasi variabel yang belum ada nilainya :
    int angka;
  2. untuk mengisi variabel yang belum terdapat nilainya dapat dilakukan dengan
    angka= 27;
  3. Membuat deklarasi variabel yang memiliki tipe data sama
    int angka,umur,nomor;

Konversi Tipe Data

  • Konversi string menjadi integer
    Integer.parseInt(namaVariabel)
  • Konversi tipe data apa saja ke tipe data int

    1. Integer.parseInt(namaVariabel)
    2. Integer.valueOf(2.1)
    3. Objek.toInt();
    4. (int) namaVariabel;
