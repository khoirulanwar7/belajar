Tutorial Pemrogaman Java untuk Pemula

Kumpulan tutorial belajar Java untuk pemula, dari dasar hingga mahir.

Pada artikel ini kita akan membahas apa itu Java? Asal Usul Java? Mengapa belajar Java? dan Bagaimana untuk memulai belajar Java?

Apa itu Java?

Java merupakan bahasa pemrograman yang digunakan secara luas untuk pengodean aplikasi web. Bahasa ini telah menjadi pilihan populer di antara developer selama lebih dari dua dekade, dengan jutaan aplikasi Java yang digunakan saat ini.

Sejarah Singkat Java.

Java dikembangkan pada tahun 1990 oleh seorang insinyur dari perusahaan Sun Microsystem yaitu James Gosling sebagai bahasa pemrograman yang berperan sebagai otak untuk peralatan pintar (seperti TV Interaktif dan Oven Serba bisa).

Bahasa ini banyak mengadopsi sintaks dari bahasa pemrograman lain seperti bahasa C dan bahasa C++, namun dengan sintaks yang modelnya lebih sederhana.

Pada saat itu James Gosling meraasa kurang puas dengan hasil yang ia peroleh ketika menulis program dengan bahasa C++ dan bahasa pemrograman lainnya, sehingga dia menulis programnya menggunakan bahasa pemrogramannya sendiri yang sesuai dengan kebutuhannya.

Nama pertama Java saat pertama kali dirilis bukanlah Java, akan tetapi adalah Oak yang kemudian berganti menjadi Green dan pada akhirnya berganti lagi menjadi Java, nama Java sendiri berasal dari nama Kopi Jawa (Java Coffee).
