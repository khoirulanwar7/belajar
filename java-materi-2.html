Tutorial Pemrogaman Java untuk Pemula

Kumpulan tutorial belajar Java untuk pemula, dari dasar hingga mahir.

Apa itu Algoritma?

1. Algoritma : urutan langkah-langkah logis penyelesaian masalah yang disusun secara Sistematis dan Logis.
2. Sistimatis : urutan langkah-langkah yang disusun harus jelas, memiliki pola tertentu.
3. Logis : masuk akal, bisa dibaca dan diterima oleh akal.
4. Bisa dinilai benar salahnya.

Struktur Algoritma.

1. Runtutan/sekuensial
Digunakan untuk program yang pernyataannya sequential atau urutan.
2. Pemilihan
Digunakan untuk program yang menggunakan pemilihan atau penyelesaian kondisi.
3. Perulangan
Digunakan untuk program yang peryataannya akan dieksekusi berulang-ulang.

Fungsi Input

1. Untuk membaca input dari keyboard menggunakan library Scanner yang di import kedalam program java.
2. Caranya dengan menuliskan perintah import.java.util.Scanner di baris paling atas
3. Setelah itu tambahkan perintah deklarasi Scanner didalam fungsi main Scanner sc = new Scanner (System.in); Untuk nama scanner bebas , bisa menggunakan sc, input ataupun yang lainnya.
4. Selanjutnya untuk menginputkan data dapat dilakukan dengan menuliskan nama scanner kemudian ditambah dengan nextTipedata() ;
