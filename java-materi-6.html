Tutorial Pemrogaman Java untuk Pemula

Kumpulan tutorial belajar Java untuk pemula, dari dasar hingga mahir.

Perulangan

1. Definisi

Perulangan (loop) adalah perintah untuk mengulang satu atau lebih statement sebanyak beberapa kali.

2. Jenis Perulangan

• Perulangan For

(digunakan pada pengulangan yang jumlah perulangannya sudah pasti atau sudah diketahui sebelumnya).
for (inisialisasi; syarat; update) statement;

Penjelasan :

o Inisialisasi tugasnya untuk menyimpan hitungan perulangan.
o Syarat merupakan kondisi yang berisi batas agar perulangan tetap dilakukan.
o Update merupakan perubahan nilai pada setiap perulangan (increment atau decrement)

Alur :

o Perulangan diawali dengan melalukan inisialisasi sekali
o Jika syarat bernilai TRUE jalankan semua statement di dalam perulangan.
o Lakukan update, lalu periksa kembali syarat.
o Jika syarat bernilai FALSE, hentikan perulangan.

• Perulangan While

Perulangan While digunakan untuk perulangan yang jumlahnya tidak diketahui sebelumnya

Inisialisasi
while (syarat){
statement 1;
statement 2;
...
increament
}

Penjelasan :

o Syarat adalah kondisi yang harus dipenuhi agar perulangan tetap dilakukan.
o Perulangan while akan terus berjalan selama syarat perulangan bernilai TRUE

• Perulangan DO-WHILE

inisialisasi
do {
statement 1;
statement 2;
.....
increament
} while (syarat)

Penjelasan :

o Perulangan do while hampir sama dengan peerulangan while, namun perintah do while akan menjalankan statement terlebih dahulu setelah itu baru diperiksa syaratnya, oleh karena itu perulangan do while setidaknya menjalankan statement satu kali, meskipun syarat perulangan bernillai FALSE.
